# Road Accident Analysis Project:

## **Overview**
This project analyzes traffic accident data in India to identify patterns, hotspots, and key factors contributing to road accidents. The analysis includes SQL queries to extract meaningful insights from accident data, and a dashboard visualization (as seen in the screenshot) that presents key metrics and trends.

## *Key Features*
Data Analysis
1)Accident Trends: Identifies states with highest accidents, time patterns, and severity distributions

2)Environmental Factors: Analyzes impact of weather, road conditions, and lighting

3)Temporal Patterns: Compares weekday vs weekend accidents and time-of-day trends

4)Severity Analysis: Examines fatal vs non-fatal accident patterns

5)Year-over-Year Comparison: Tracks changes in fatality rates between 2022-2023

## *Dashboard* 
1)Filter Panel: Allows filtering by state, vehicle type, and year

2)Key Metrics: Displays total casualties, accidents, alcohol-related incidents

## *Visualizations:*

1)Casualties by road type

2)Accident severity breakdown

3)Lighting condition impacts

## **SQL Highlights**
The SQL script includes 15 analytical objectives that:

1)Calculate total casualties and fatalities

2)Identify high-risk states and locations

3)Analyze temporal patterns (day of week, time of day)

4)Examine environmental factors (weather, road conditions)

5)Compare severity patterns across states

6)Track year-over-year changes
